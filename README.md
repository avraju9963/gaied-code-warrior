# 📌 AI Email Parser

This project is an **AI-powered email parsing system** that processes `.eml` files, extracts text from emails and attachments, summarizes the content, and forwards parsed data to another service.

---

## 🚀 Features
✅ Parse `.eml` email files  
✅ Extract **text** from email body and attachments  
✅ Supports multiple attachment types:  
   - 📄 **PDFs**  
   - 🖼 **Images (JPG, PNG, etc.)** (OCR-based extraction)  
   - 📝 **DOCX (Word documents)**  
   - 📑 **JSON, XML, and CSV files**  
✅ AI-powered **text summarization**  
✅ Automatically **forwards** parsed email data to a target endpoint
